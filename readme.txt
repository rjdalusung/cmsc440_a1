To use this program, first open ChatServer.py by running

python ChatServer.py {PORT}

where port is the port number wanted, this can only be between 10000-11000


Next, open up the client using

python ChatClient.py {HOSTNAME} {PORT} {NICKNAME} {ID}

if there are errors, the client will tell you before exiting. Keep in mind that nicknames and ID's must be unique


To stop execution for the server, press CTRL + C

To stop execution for the client, type "disconnect"
